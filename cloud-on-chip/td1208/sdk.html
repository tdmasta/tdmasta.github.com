Overview Boards of a feather...

Description

The TD1208 SDK provides a development and demonstration platform for Telecom Design TD1208 SIGFOX™ gateway modules and software tools.

This overview focuses on the TD1208 evaluation board as a development platform for the TD1208 SIGFOX™ Gateway Module.

Get your EVB

Board features

TD1208 SIGFOX™ Gateway module in LGA25 package

  • SIGFOX™ certified
  • Frequency range = ISM 868 MHz
  • Receiver sensitivity =-126 dBm
  • (G)FSK, 4(G)FSK, GMSK / OOK Modulation
  • Max output power : +14 dBm
  • Low active radio power consumption : 13/16 mA RX , 37 mA TX @ +10 dBm
  • Power supply = 2.3 to 3.3 V
  • LGA25 (25.4×12.7×3.81mm) Land Grid Array package
  • Up to 12 GPIOs pins
  • I2C bus interface
